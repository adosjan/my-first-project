## Топ бесплатных фреймворков для новичка

### 1. LangChain ⭐ РЕКОМЕНДУЕТСЯ

| Параметр | Значение |
|----------|----------|
| Цена | **Бесплатно** (open source) |
| Сложность | 6/10 |
| Языки | Python, TypeScript/JavaScript |
| Документация | Отличная |

**Что это:**
"Швейцарский нож" AI-агентов. Самый популярный фреймворк в 2025. Модульный подход — полный контроль над поведением агента.

**Плюсы:**
- Огромное количество интеграций
- Много примеров в комьюнити
- Работает с Python и JavaScript

**Минусы:**
- Сложный для новичка — много компонентов
- Требует LangGraph для сложных workflows

**Когда использовать:**
Для production-систем, когда нужен полный контроль.

---

### 2. CrewAI ⭐ ЛУЧШИЙ ДЛЯ СТАРТА

| Параметр | Значение |
|----------|----------|
| Цена | **Бесплатно** (core), Cloud от $99/мес |
| Сложность | 4/10 |
| Языки | Python |
| Документация | Хорошая |

**Что это:**
Платформа для создания "команд" агентов. Каждый агент имеет свою роль, и они работают вместе над задачей.

**Плюсы:**
- Очень просто начать
- Отлично для multi-agent систем
- Минимум кода для запуска

**Минусы:**
- Ограничен для сложных workflows
- Меньше гибкости чем LangChain

**Когда использовать:**
Для контент-пайплайнов, исследовательских задач, командной работы агентов.

---

### 3. AutoGPT

| Параметр | Значение |
|----------|----------|
| Цена | **Бесплатно** (open source) |
| Сложность | 5/10 |
| Языки | Python |
| Документация | Средняя |

**Что это:**
"Дикий ребёнок" AI-агентов. Даёшь цель — он сам планирует и выполняет шаги.

**Плюсы:**
- Полная автономность
- Отлично для экспериментов
- Активное комьюнити

**Минусы:**
- Не готов для production
- Может "зациклиться" на задачах
- Непредсказуемое поведение

**Когда использовать:**
Для прототипов и экспериментов. Не для продакшена.

## Рекомендация для новичка с $0

### Вариант 1: Максимально простой (no-code)
```
Ollama (локальные LLM)
    +
Dify или Flowise (визуальный редактор)
    =
100% бесплатная платформа для AI-агентов
```

### Вариант 2: С изучением Python
1. **Этап 1:** CrewAI — простой синтаксис, быстрый результат
2. **Этап 2:** LangChain — для сложных систем и production

### Бесплатные LLM для агентов
- **Groq** — бесплатный, быстрый (llama-3.3-70b) ✅ Уже используем
- **Together AI** — бесплатный tier
- **Ollama** — локальные модели (100% бесплатно, нужен мощный ПК)
- **Google AI Studio** — бесплатный доступ к Gemini

### Требования для локальных LLM (Ollama)
- RAM: 8 GB (16 GB рекомендуется)
- Диск: 10-20 GB
- GPU: необязательно, но ускоряет

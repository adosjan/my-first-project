## Платные сервисы

### 1. Exploding Topics
- **Цена:** $39-249/месяц
- **Что даёт:** Растущие темы до того, как они стали мейнстримом
- **Источники:** Поисковые запросы, социальные сети, форумы
- **API:** Нет публичного API
- **Фишка:** Показывает тренды за 6-12 месяцев до пика

### 2. Trend Hunter
- **Цена:** Бесплатный доступ к части контента, Pro от $199/мес
- **Что даёт:** Тысячи микро и макро-трендов в потребительском секторе
- **Источники:** Собственные аналитики, краудсорсинг
- **Фокус:** Продуктовые тренды, бренды, consumer behavior
- **Ограничение:** Не даёт keyword-данные для SEO

### 3. BuzzSumo
- **Цена:** от $199/месяц
- **Что даёт:** Вирусный контент, trending темы
- **Источники:** Социальные сети, engagement метрики
- **API:** Есть
- **Фишка:** Ловит тренды по скорости распространения

### 4. Treendly
- **Цена:** Платный
- **Что даёт:** Автоматический поиск трендов + прогнозирование
- **Фишка:** Предсказывает будущую траекторию тренда

## Бесплатные альтернативы ⭐

### 1. Google Trends — ОБЯЗАТЕЛЬНО
- **Цена:** Бесплатно
- **Что даёт:** Относительный интерес к поисковым запросам (0-100)
- **Плюсы:** Real-time данные, сравнение регионов, YouTube тренды
- **Минусы:** Не показывает абсолютные объёмы поиска
- **API:** Неофициальный (pytrends)
- **Автоматизация:** ✅ Да

### 2. Glimpse (Free tier)
- **Цена:** 10 бесплатных поисков/месяц, Pro от $99
- **Что даёт:** Chrome-расширение, добавляет к Google Trends абсолютные объёмы
- **Пользователи:** Amazon, Coca-Cola, IKEA, Chanel
- **Рейтинг:** 4.9 звёзд, 150,000+ пользователей

### 3. Answer Socrates
- **Цена:** Бесплатный tier + платный
- **Что даёт:** Trending темы + вопросы, которые люди задают
- **Уникальность:** Показывает не только ЧТО растёт, но и КАК люди об этом спрашивают

### 4. Pinterest Trends — НЕДООЦЕНЁН
- **Цена:** Бесплатно
- **Что даёт:** Тренды в визуальном контенте
- **Идеально для:** DTC-продукты, e-commerce, контент-идеи

### 5. Reddit
- **Цена:** Бесплатно
- **API:** Да (бесплатный с лимитами)
- **Что даёт:** Раннее обнаружение трендов в нишевых комьюнити

### 6. Product Hunt
- **Сайт:** producthunt.com
- **Цена:** Бесплатно
- **API:** Да (GraphQL, бесплатный)
- **Что даёт:** Новые продукты, стартапы, tech-тренды

### 7. Hacker News
- **Сайт:** news.ycombinator.com
- **Цена:** Бесплатно
- **API:** Да (Firebase, бесплатный)
- **Что даёт:** Tech-тренды, стартап-идеи, обсуждения

### 8. GitHub Trending
- **Сайт:** github.com/trending
- **Цена:** Бесплатно
- **API:** GitHub API (бесплатный с лимитами)
- **Что даёт:** Trending репозитории, языки, технологии

## Как построить бесплатную систему мониторинга

### Наш Trend Hunter уже делает это:
1. **Google Trends RSS** — ежедневные тренды
2. **Reddit JSON API** — горячие посты из бизнес-сабреддитов
3. **Groq AI** — анализ и генерация бизнес-идей

### Что можно добавить (бесплатно):
- **Twitter/X API** — ограниченный бесплатный доступ
- **HackerNews API** — бесплатный, тех-тренды
- **Product Hunt RSS** — новые продукты

### Ключевой вывод
> Бесплатные инструменты дают базовые данные — интерпретировать нужно самому.
> Платные дают early prediction и готовый анализ.
> **Решение:** Использовать бесплатные источники + AI для анализа = конкурентное преимущество.
